# Shakti Notes (Notes App)

It's a intresting Notes app where you can write your task and track them you can do opeartions like create , read , update , delete

## Features
- Create, edit and delete notes
- Categories (Study, Personal, College, Ideas, Meetings, etc.)
- Local persistence using localStorage
- Dark/light friendly CSS

## Tech
- React
- CSS
- Javascript
- Browser localStorage for persistence

## Install (Windows)
1. Open a terminal in the project folder:
   Clone the project 
2. Install dependencies:
   npm install
3. Run 
   npm run dev
